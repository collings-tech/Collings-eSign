# Signature fonts for PDF embedding

Place the following TTF files here so the signed PDF uses the same font style as the one chosen in the app:

| Font name (from app) | Filename |
|----------------------|----------|
| Dancing Script | `DancingScript-Regular.ttf` |
| Great Vibes | `GreatVibes-Regular.ttf` |
| Allura | `Allura-Regular.ttf` |
| Pacifico | `Pacifico-Regular.ttf` |
| Sacramento | `Sacramento-Regular.ttf` |

Download the regular/400 weight from [Google Fonts](https://fonts.google.com/) (each font page has a "Download family" option). Extract the `.ttf` and rename if needed to match the table above.

If a file is missing, the PDF will fall back to the default system font for typed signatures.
